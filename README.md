# NetCop

NetCop — это мощная утилита для мониторинга сети и шейпинга трафика (ограничения скорости) на уровне процессов для Windows. Состоит из скрытого агента, работающего на целевых машинах, и центрального сервера с веб-дашбордом для управления.

## Основные возможности

- **Детальный мониторинг:** Сбор метрик (CPU, RAM, активные соединения) и отображение топ-10 процессов, потребляющих трафик.
- **Traffic Shaping (IN+OUT):**
  - Ограничение **исходящего** трафика через политики `PowerShell NetQosPolicy`.
  - Ограничение **входящего** трафика через драйвер `WinDivert` и кастомный алгоритм Token Bucket (дроп пакетов для естественной отработки TCP).
- **Массовое управление:** Группировка процессов по категориям (игры, торренты, браузеры) для применения лимитов в один клик.
- **System Tray Agent:** Агент может работать как фоновый процесс без консоли с индикатором в системном трее Windows.
- **Персистентность:** Все установленные лимиты надежно сохраняются в локальной SQLite базе данных сервера.

## Архитектура

- **Сервер (`server/`):** FastAPI, SQLite (для хранения лимитов), веб-дашборд на чистом HTML/JS/CSS (Chart.js).
- **Агент (`agent/`):** Python-скрипт (`psutil`, `pydivert`, `pystray`), опрашивающий ОС и получающий команды от сервера через long-polling.

## Требования

- Windows 10/11 (Агент работает только на Windows).
- Python 3.8+ (для запуска из исходников).
- **Права Администратора** для запуска Агента (необходимы для `WinDivert` и политик QoS).

## Установка и запуск (из исходников)

### Сервер
1. Перейдите в папку `server/`.
2. Установите зависимости: `pip install -r requirements.txt`.
3. Запустите сервер: `python main.py` (или `launcher.py`).
4. Дашборд будет доступен по адресу `http://127.0.0.1:8000` (или по вашему IP).

### Агент
1. Перейдите в папку `agent/`.
2. Запустите `install.bat`. Скрипт установит зависимости и спросит:
   - IP сервера (по умолчанию `http://127.0.0.1:8000`).
   - Секретный API ключ (по умолчанию `secret`).
   - Включить ли WinDivert шейпер (потребует скачивания драйвера).
   - Запускать ли в системном трее (без окна).
3. Агент запустится автоматически после настройки. **Обязательно запускайте от имени Администратора.**

## Сборка в `.exe` (PyInstaller)

Проект полностью готов к сборке в единые бинарные файлы:

- **Сборка сервера:** Запустите `server/build_server.bat`. Готовый файл появится в `server/dist/NetCopServer.exe`.
- **Сборка агента:** Запустите `agent/build_agent.bat`. Файл появится в `agent/dist/NetCopAgent.exe`. 
*(Внимание: если вы используете WinDivert, вам потребуется положить файлы драйвера `WinDivert.dll` и `WinDivert64.sys` рядом с exe-файлом агента, либо добавить их при сборке через `--add-binary`).*

## Безопасность

Связь между агентами и сервером (а также доступ к API) защищена с помощью общего секретного ключа (`NETCOP_API_KEY`), который передается в заголовке `X-NetCop-Key`. Убедитесь, что ключ совпадает на клиенте и сервере.
